This is a system for administrating storage, kitchen and two cafeteria. The system has an sqlite database with several tables which reflect the state and planned activity in the kitchen. The system has an app running as web service to be used by the kitchen, cafeteria and administrator. Kitchen, cafeteria and administrator roles open corresponding flaps of the app.

Description of the data tables are in DATABASE.md file.

Description of the functions necessary to operate database tables are in DATA_FUNCTIONS.md

Description of the app is in APP.md

The project is written in python and bash for scripting.

The project contains a folder with setup scripts, for setting up an SQLite database.

Another folder contains a python module with functions for operating tables in the database.
